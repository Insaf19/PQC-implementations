
The package Compact-LWE-Public-Key_Enc inlcudes four directories:

1) Supporting_Documentation
   --"cover-page.pdf" is the Cover Page file.
   --"specification-compact-lwe.pdf" is the specification of Compact-LWE public key encryption scheme.


2) Reference_Implementation
   -- All source files of reference implementation.
   -- Run "make" to generate PQCgenKAT_encrypt.
   -- Run "PQCgenKAT_encrypt" to generate KAT files PQCencryptKAT_232.req and PQCencryptKAT_232.rsp 
       (tested on 64-bit Ubuntu 17.10 and 32-bit Ubuntu 14.04).
   -- "make clean" is supported. 


3) Optimized_Implementation
   -- All source files of optimized implementation.
   -- Run "make" to generate PQCgenKAT_encrypt.
   -- Run "PQCgenKAT_encrypt" to generate the same KAT files PQCencryptKAT_232.req and PQCencryptKAT_232.rsp (tested on 64-bit Ubuntu 17.10).
   -- "make clean" is supported.
   -- Run "make perf" to generate "perf".
   -- Run "perf" to repeat the performance test described in the specification (tested on 64-bit Ubuntu 17.10).
   
2) KAT
   -- PQCencryptKAT_232.req and PQCencryptKAT_232.rsp are KAT files generated by PQCgenKAT_encrypt as described above.
